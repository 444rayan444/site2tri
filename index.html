Os três principais comandos para se criar em um site

esses são os três comandos principais para criar um site web

STYLE CSS é um mecanismo para adicionar estilos a uma página web, aplicado diretamente nas tags HTML ou ficar contido dentro das tags style

INDEX HTML é uma linguagem de marcação utilizada na construção de páginas na Web.

JAVA SCRIPT é uma linguagem de programação interpretada estruturada, de script em alto nível com tipagem dinâmica fraca e multiparadigma.
